# それぞれのファイルの役目
ファイルの役目を綴ろう

## res/values/strings.xml
→アプリ内で画面表示する文字列は、原則このxmlファイルに定義してあげること。

## res/layout/activity_main.xml
→初期画面用のレイアウトファイル  
・ConstraintLayout：柔軟な画面が作成できる一方、ある程度操作になれている必要があるよ  
・LinearLayout：入門者にも扱いやすい？
